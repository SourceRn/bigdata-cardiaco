# Sistema de Predicción de Enfermedades Cardíacas

Proyecto desarrollado en Python orientado al análisis y predicción de enfermedades cardíacas mediante técnicas de Machine Learning.

El sistema implementa dos modelos:

- Regresión Logística
- Random Forest

El objetivo principal es comparar el desempeño de ambos modelos utilizando métricas de clasificación médica.

---

# Objetivos del Proyecto

- Analizar datos médicos de pacientes.
- Aplicar técnicas de preprocesamiento de datos.
- Entrenar modelos de Machine Learning.
- Evaluar precisión y capacidad predictiva.
- Comparar distintos enfoques de clasificación.
